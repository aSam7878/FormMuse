# Using FormMuse with Agents

Coding agents should inspect the target project's React, TypeScript, Tailwind CSS 4, and shadcn Base UI setup before installing a FormMuse template.

Agents should preserve adopter-owned code, avoid unsupported Radix migration, and keep template changes inside the installed source boundary unless the owner asks for broader integration.
