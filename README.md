# FormMuse

FormMuse is an open-source collection of premium, ready-to-paste Form Templates for React applications. Templates include polished visual composition, accessible form behavior, animation, validation, and a provider-agnostic asynchronous `onSubmit` contract.

The project is currently under active development toward its V1 launch collection: ten contact forms, five newsletter forms, and five quote-request forms.

## Status

FormMuse is not ready for public installation yet. The first vertical slice is the Hanging Gifts Contact Form Template.

## V1 baseline

- React, TypeScript, and Tailwind CSS 4.
- shadcn controls backed by Base UI.
- React Hook Form and Zod.
- Portable Next.js and Vite React template source.
- Static Next.js website, documentation, registry, and previews.
- MIT-licensed FormMuse-authored software and documentation.

See [FormMuse Tech Stack.md](./FormMuse%20Tech%20Stack.md), [reference.md](./reference.md), and [the implementation plan](./docs/implementation-plan.md) for the locked architecture.
