# Use existing agent tooling

Every Form Template page will provide a Copy Agent Prompt action backed by clear machine-readable registry metadata, well-named exported Zod schemas and value types, and a typed Submission Connection. FormMuse will remain compatible with the existing shadcn MCP server rather than building a custom FormMuse MCP server, AI chat interface, or hosted agent backend in V1. This gives adopters a predictable Agent Handoff while keeping FormMuse focused on authoring and distributing Form Templates instead of operating AI infrastructure.
